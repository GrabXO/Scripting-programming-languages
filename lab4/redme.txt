Перед тем, как запустить основной файл "lab4.py", необходимо установить библиотеку Python PyQt5, установить новую версию Python (на момент написания кода последняя версия - 3.13.0) и скачать дополнительный файл "lab4.db".
Чтобы установить библиотеку PyQt5, в командной строке нужно прописать:
  pip install PyQt5
Все скачанные файлы поместить в одну папку и запустить файл "lab4.py", воспользовавшись IDE PyCharm или VS Code.
