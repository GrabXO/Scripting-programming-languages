Перед тем, как запустить файл, необходимо установить библиотеку Python requests и новую версию Python (на момент написания кода последняя версия - 3.12.6).
Чтобы установить библиотеку requests, в командной строке нужно прописать:
  pip install requests
После этого, можно запускать файл при помощи IDE PyCharm или VS Code.
