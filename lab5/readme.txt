Перед тем, как запустить основной файл lab5.py, необходимо установить библиотеку Python PyQt5 и aiohttp, установить новую версию Python (на момент написания кода последняя версия - 3.13.0) и скачать дополнительный файл lab5.db.
Чтобы установить библиотеку PyQt5, в командной строке нужно прописать:
  pip install PyQt5
  pip install aiohttp
Все скачанные файлы поместить в одну папку и запустить файл lab5.py, воспользовавшись IDE PyCharm или VS Code.
