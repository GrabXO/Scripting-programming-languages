Перед тем, как запустить основной файл lab6.py, необходимо установить библиотеку Python PyQt5 и matplotlib, установить новую версию Python (на момент написания кода последняя версия - 3.13.0) и скачать дополнительный файл sample_data.csv.
Чтобы установить библиотеку PyQt5 и matplotlib, в командной строке нужно прописать:
  pip install PyQt5
  pip install matplotlib
Все скачанные файлы поместить в одну папку и запустить файл lab6.py, воспользовавшись IDE PyCharm или VS Code.
